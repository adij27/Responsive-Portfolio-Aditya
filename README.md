# Aditya Jadhav — Portfolio

Personal portfolio website built with HTML, CSS, and JavaScript (Vite). Showcases data analytics projects, work experience, and technical skills.

**Live site:** https://adij27.github.io/Responsive-Portfolio-Aditya

---

## Projects Featured

- **NYC Citi Bike Analytics Pipeline** — dbt + BigQuery + Looker Studio pipeline on 61.7M records
- **Supply Chain KPI Dashboard** — Python/Tableau pipeline tracking 10+ operational KPIs (Olam Food Ingredients)
- **Text Summarization API** — FastAPI + Hugging Face Transformers REST API

---

## Tech Stack

| Layer | Tool |
|-------|------|
| Framework | Vite |
| Markup | HTML5 |
| Styling | CSS3 (custom properties, Grid, Flexbox) |
| Interactivity | Vanilla JavaScript |
| Animations | ScrollReveal |
| Icons | Remixicons |
| Contact form | EmailJS |

---

## Run Locally

```bash
git clone https://github.com/adig27/Responsive-Portfolio-Aditya.git
cd Responsive-Portfolio-Aditya
npm install
npm run dev
```
